Kostprobe?

You collect a virtual drink travelling a digestive tract? It might taste excitingly different in the real world.

“KOSTPROBE?” is an interactive object that can be used as a one-click drinking automat. The virtual environment reflects in real time the 360 view of the automat’s insides. In the moment of drinking the warm, yellow liquid the view changes to an abstract, expanded reality.
